# CryptoGuide 🔥
### Binance & Crypto Assistant — Powered by Google Gemini (Free)

---

## Deploy to Vercel (Free)

### Step 1 — Upload to GitHub
1. Go to https://github.com and create a free account
2. Click **New Repository** → name it `cryptoguide` → Create
3. Upload all these files to the repo (drag & drop)

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com → sign up with GitHub
2. Click **Add New Project** → Import your `cryptoguide` repo
3. Click **Deploy** ✅ (no env variables needed!)

### Step 3 — Get your free Gemini API Key
1. Go to https://aistudio.google.com
2. Sign in with Google
3. Click **Get API Key** → **Create API key** → Copy it

### Step 4 — Use the app
- Open your Vercel URL
- Paste your Gemini API key when prompted
- Start chatting!

### Step 5 — Add to Phone Home Screen
- Open your Vercel URL in Safari (iPhone) or Chrome (Android)
- Tap **Share** → **Add to Home Screen** 📱

## Local Development

```bash
npm install
npm start
```

No .env needed — just paste your Gemini key in the app when it asks.
